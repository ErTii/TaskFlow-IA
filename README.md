# TaskFlow-IA

TaskFlow AI est une application intelligente qui analyse automatiquement les emails et les transcriptions de réunion pour en extraire les tâches, les prioriser, les assigner et les centraliser dans Notion, avec un dashboard de suivi et une proposition de création de réunions dans Google Calendar.

https://taskflow-ia.streamlit.app/

Version de l'application :

# V0.0.01 : 
- une saisie manuelle
- un email scanné
Envoyer dans Notion avec :
- tâche
- priorité
- assigné

# V0.0.02 : 
- validation avant envoi
- statuts des tâches
- suppression de l’affichage après envoi
- dashboard connecté à Notion
- filtres
- Camembert de répartition
  
# V0.0.03 : 
- détection d’une demande de réunion dans un mail ou un texte
- proposition de création d’un événement
- ajout des participants
- création dans Google Calendar en local

# V0.0.04 : 
- modification du statut des tâches directement depuis l’application
